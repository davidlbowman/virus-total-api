# VirusTotal Directory Scanner

A TypeScript utility that scans directories for malware using the VirusTotal API.

## Features

- Recursive directory scanning
- Support for both free and paid VirusTotal API tiers
- Automatic rate limiting for free tier
- Summary report of scan results

## Setup

1. Install dependencies
2. Set your VirusTotal API key
3. Set the directory to scan
4. Run the script

```bash
bun run start
```
